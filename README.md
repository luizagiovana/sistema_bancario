# Sistema Bancário

Projeto desenvolvido em Python para simular operações bancárias básicas através do terminal.

## Funcionalidades

- Depósito
- Saque
- Controle de limite diário de saques
- Extrato de movimentações
- Controle de saldo

## Tecnologias utilizadas

- Python 3

## Como executar

Clone o repositório:

```bash
git clone <url-do-repositorio>
```

Acesse a pasta:

```bash
cd sistema-bancario
```

Execute o programa:

```bash
python main.py
```
